Frequently asked questions about QuickBooks
and the Answers

How do I recover user ID and reset my QuickBooks account password?

Answer- You can easily get the best results for your needs by following the following steps:

  1. Open Intuit website > click on forgot user ID or password option. Then enter your email address and user ID and continue
  2. Follow the on-screen instructions carefully. When the account is verified using the above user ID or email address, you get an email for resetting the password.

How do I run payroll online using QuickBooks?

Answer- You can easily get the best results for your needs by following the following steps-

  1. Go to payroll online and click on dashboard
  2. Click run payroll and fill information
  3. Review information and submit
  4. Follow the given instructions on the screen to print out the payroll checks

How to upgrade a company file in the QuickBooks?

Answer- You can easily upgrade a company file in the QuickBooks by following the following steps -

  1. Open the QuickBooks
  2. Click file and open or restore company file
  3. Select open a company file
  4. Browse the company file that you want to upgrade
  5. Click open
  6. Now upgrade a new version of company file in the QuickBooks
